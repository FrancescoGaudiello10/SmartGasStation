## Esecuzione (Rails 2.3.0)

### 1. Installare **dipendenze**

`bundle install`

### 2. **Creare** il database **aggiornato** ad oggi tramite script automatico 

Aprire terminale in `/SmartGasStation/db` ed eseguire `./generateUpdatedDB.sh`


### 3. Eseguire **server**

`rails server`
